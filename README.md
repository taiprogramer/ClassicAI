# ClassicAI

Solution for classic AI problems written in C

## Run

I have written a script called `run.sh` to help you easily run the demo.

On Linux, simply run:

```sh
chmod +x run.sh
./run.sh [FOLDER_NAME]
```

Example:

```sh
./run.sh WaterJug
```
